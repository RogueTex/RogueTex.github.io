ChargePilot - EV Charger Expansion Optimizer

Python | ETL pipelines | Ridge regression | Graph message passing | Facility-location optimization | Next.js

Tab Graveyard - Chrome Extension

Manifest v3 | Vanilla JS | Chrome local storage | Groq/OpenRouter classification | Local-first UX

Local-first Chrome extension that captures closed tabs into a searchable graveyard, with filter, resurrect, and delete flows for large personal tab datasets. Optional LLM classification runs through Groq or OpenRouter without requiring a backend and degrades gracefully when no API key is configured.

  • Shipped a product-facing Manifest v3 extension with fast local search and filter UX on stored tab data.

2A1B Solver - Reinforcement Learning for Alphabet 2048

Python | Gymnasium | PyTorch Double DQN | Dueling/prioritized replay | Playwright

Reinforcement-learning stack for Alphabet 2048 that pairs a Gymnasium environment with a PyTorch DQN agent, evaluation tooling, and a browser bridge for live play. The system uses a Double DQN training loop with optional dueling and prioritized replay, then deploys policy decisions into the public game UI through Playwright.

  • Built a full RL codebase with environment, agent, training scripts, and diagnostics instead of a one-off notebook workflow.
  • Closed the model-to-integration loop by automating gameplay in a real browser session for realistic evaluation.

ReceiptOps Agent - Document ML Pipeline

LangGraph | OCR | LayoutLMv3 | Document understanding | Human-in-the-loop approvals

LangGraph-powered document ML pipeline for receipt automation that combines OCR, LayoutLMv3-based document understanding, and anomaly-aware validation routing. It extracts structured receipt fields, sends low-confidence cases through human-in-the-loop approvals, and uses feedback signals to improve downstream rules and quality over time.

  • Unified extraction, validation, and confidence-based routing in one workflow instead of relying on a single-model pass.
  • Captured reviewer feedback to iteratively improve approval thresholds, validation logic, and output reliability.
