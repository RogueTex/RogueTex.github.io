Tab Graveyard - Chrome Extension

Manifest v3 | Vanilla JS | HTML/CSS | Chrome Storage | Groq/OpenRouter

Local-first Chrome extension that captures closed tabs into a searchable graveyard, with filter, resurrect, and delete flows designed for heavy personal tab history.
